## 项目架构

| 项目         | 配置信息                                 |
| ------------ | ---------------------------------------- |
| 应用包名     | `com.fongmi.android.tv`                  |
| 最低适配版本 | 24（安卓7.0）                            |
| 支持架构     | `arm64-v8a`、`armeabi-v7a`               |
| 编译变体     | `leanback`（电视版）、`mobile`（手机版） |

```
TV/
├── app/            主程序（内置两套界面适配方案）
├── catvod/         爬虫底层封装（爬虫接口、网络请求框架）
├── quickjs/        QuickJS JavaScript脚本引擎
├── chaquo/         Chaquopy Python脚本引擎
```

`app/src/main/` 存放双端通用业务逻辑，`app/src/leanback/` 与 `app/src/mobile/` 分别编写电视端、手机端专属界面。

## 播放器

- **核心内核**：Media3(ExoPlayer) + FFmpeg软解码，自动切换硬解/软解模式
- **画面渲染**：支持SurfaceView、TextureView两种渲染方式
- **加密播放**：兼容Widevine、PlayReady、ClearKey加密源，支持`#KODIPROP`格式密钥
- **弹幕功能**：集成DanmakuFlameMaster弹幕框架，精准同步播放进度，支持远程推送弹幕
- **字幕支持**：加载SRT/SSA/ASS外挂字幕，适配系统字幕服务，可远程实时注入字幕
- **实用功能**：播放倍速、多比例缩放、画中画播放、后台音频播放、片头片尾自动跳过

---

## 点播功能

- 多资源站点分类浏览，支持按年份、地区、题材等条件筛选
- 多站点并行搜索，关键词自动简繁转换提升搜索适配性
- 播放失败智能换源：优先切换解析线路→更换播放源→跨站点检索资源
- 本地观影记录（留存60天）、资源收藏、无痕浏览模式
- 电视端适配遥控器操作；手机端支持手势调节亮度/音量/播放进度，滑动切换剧集，屏幕旋转与锁屏锁定

---

## 直播功能

- 兼容M3U、TXT（`#genre#`分组格式）、JSON主流直播源格式
- **节目预告**：支持XMLTV格式EPG节目单，兼容.gz压缩包，每6小时自动更新
- **回看时移**：适配append、pltv等多种回看直播协议
- 频道收藏管理，支持分组加密隐藏
- 内置专属直播引擎：TVBus、ForceTech

---

## 爬虫引擎

支持三种编程语言编写资源爬虫：

- Java 程序包（DexClassLoader动态加载）
- JavaScript 脚本（QuickJS引擎运行）
- Python 脚本（Chaquopy引擎运行）

通过`api`字段指定调用爬虫脚本，`ext`字段传入初始化配置参数，完整接口规范查看 [SPIDER.md](docs/SPIDER.md)。

---

## 网络功能

- **安全DNS**：支持HTTPS加密DNS解析，可自定义优选解析地址
- **全局代理**：兼容HTTP/HTTPS/SOCKS4/SOCKS5代理，可按域名规则智能分流
- **本地域名**：自定义DNS解析规则，支持通配符批量匹配
- **跨域修复**：根据域名规则自动添加自定义跨域响应请求头
- **广告拦截**：内置域名黑名单，自动拦截广告域名请求
- **网页嗅探**：依托正则表达式抓取网页内媒体播放地址，支持自定义浏览器标识伪装

---

## DLNA投屏

- **投屏发送端**：手机端可扫描局域网内DLNA设备，推送视频影音资源投屏播放
- **投屏接收端**：电视端作为投屏接收设备，接收其他设备推送的媒体内容

基于Cling 2.1.1 UPnP协议开发，支持播放、暂停、停止、进度拖拽全操控。

---

## 远程控制

应用启动后自动本地搭建NanoHTTPD轻量网页服务，端口从**9978**自动遍历至**9998**，可实现远程播放控制、字幕弹幕推送、多设备播放同步等操作，完整接口文档查看 [LOCAL.md](docs/LOCAL.md)。

---

## 配置说明

影音主配置为程序核心入口，可通过网络链接或本地文件路径加载，核心配置字段：

- 点播资源站点（`sites`）、视频解析规则（`parses`）
- 直播资源源地址（`lives`）
- 网络全局设置（加密DNS、代理、域名规则、广告拦截）

直播配置可内嵌在主配置内，也可单独独立配置加载，全部配置参数详解查看 [CONFIG.md](docs/CONFIG.md)。
